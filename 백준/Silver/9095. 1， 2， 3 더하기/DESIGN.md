1의 경우 : 1( = 1)

2의 경우 : 2( = 2, 1 + 1)

3의 경우 : 4( = 3, 2 + 1, 1 + 2, 1 + 1 + 1)

이후 4부터 dp 적용

  - 현재 자연수( = i)를 만들 수 있는 방법
    - (i - 3) 경우에서 3을 더함 : 4가지
    - (i - 2)에는 (i - 3)을 만드는 경우도 포함되기 때문에 (i - 2)에서 (i - 3)을 뺀 경우에서 2를 더함 : 2가지
    - (i - 1)에는 (i - 2), (i - 3)을 만드는 경우도 포함되기 때문에 (i - 3)에서 (i - 2)와 (i - 1)을 뺀 경우에서 1을 더함 : 1가지
